# Ikoner för PWA och App

Placera följande ikoner här:

- `icon-192.png` (192x192)
- `icon-512.png` (512x512)
- `maskable-512.png` (512x512, maskable icon)

För att generera alla nödvändiga ikoner och splash screens för iOS/Android:

```bash
npx capacitor-assets generate
```

Detta läser från `/resources` mappen och genererar alla nödvändiga storlekar.
